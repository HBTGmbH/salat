Export

Download all working day data and time bookings for a month as CSV.

CSV – UTF-8, comma-separated, first row is header

CSV Format

Each row represents one time booking. Working day fields repeat for every booking on the same day. Days with no bookings appear as a single row with booking columns left empty.

Export delimiter: ,  ·  Import: auto-detected from first row, supported: ,  ;  \t  |  :

Date format: yyyy-MM-dd (import also accepts dd.MM.yyyy)  ·  Time format: HH:mm (import also accepts HH:mm:ss, seconds ignored)

# Column Format Example Import Notes
0 date YYYY-MM-DD 2026-06-02 Required Working day date (ISO 8601)
1 type WORKED · NOT_WORKED WORKED Required Working day type; empty if not set
2 startTime HH:mm 08:00 If WORKED Day start time; empty if not set
3 breakTime HH:mm 00:30 If WORKED Break duration; empty if not set
4 employeeorderId integer 78 One of 4/7 Order ID; if absent, suborderSign is used
5 orderSign text ALPHA Ignored Project key
6 orderLabel text Alpha Projekt Ignored Project name
7 suborderSign text DEV-BACKEND One of 4/7 Task key; used to look up the order if employeeorderId is absent
8 suborderLabel text Backend Development Ignored Task name
9 workingTime HH:mm 03:00 If booking Booked duration
10 comment text PROJ-123 initial setup Optional Work description; quote with " if it contains commas
