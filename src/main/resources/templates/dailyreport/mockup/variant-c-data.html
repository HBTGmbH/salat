CSV — compatible with import format

CSV Format Reference

Import and export use the same CSV structure. The first row is a header. Columns:

Column Format Example Notes
date YYYY-MM-DD 2026-06-02 ISO 8601
start HH:MM 08:00 Working day start time
break HH:MM 00:30 Total break duration
type WORKED | NOT_WORKED WORKED Day type

Example row: 2026-06-02,08:00,00:30,WORKED
