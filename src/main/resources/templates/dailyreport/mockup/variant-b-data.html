Overview Calendar

Import

Upload a CSV file to load working day data and time bookings. The file must use the format described below.

Days outside your contract period are skipped.

Export

Download all working day data and time bookings for a month as CSV. The file can be re-imported or processed in a spreadsheet.

CSV — UTF-8, comma-separated, first row is header

CSV Format

Each row represents one time booking. Working day fields (start, break, type) repeat for every booking on the same day. Days with no bookings (e.g. NOT_WORKED) appear as a single row with booking columns left empty.

# Column Format Example Notes
0 date YYYY-MM-DD 2026-06-02 Working day date (ISO 8601)
1 start HH:mm 08:00 Day start time; empty if not set
2 break HH:mm 00:30 Break duration; empty if not set
3 type WORKED · NOT_WORKED WORKED Working day type; empty if not set
4 employeeorderId integer 78 Internal ID of the employee order; empty for NOT_WORKED rows
5 orderSign text ALPHA Project key
6 orderLabel text Alpha Projekt Project name
7 suborderSign text DEV-BACKEND Task key
8 suborderLabel text Backend Development Task name
9 hours integer 3 Booked hours
10 minutes 0 · 15 · 30 · 45 30 Booked minutes in addition to hours
11 comment text PROJ-123 initial setup Work description; quote with " if it contains commas
12 training true · false false Whether this booking counts as training time
